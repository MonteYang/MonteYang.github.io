Python-numpy和list中的浅拷贝和深拷贝

Posted on 2019-12-16 Edited on 2020-01-08 In Python

numpy 中

  • np.copy() 方法是 深拷贝；
  • 切片操作 是 浅拷贝；
  • 直接赋值给另一个变量是 浅拷贝。

对比：Python的list中

  • 切片操作 是 深拷贝
  • 直接复制给另一个变量是 浅拷贝
