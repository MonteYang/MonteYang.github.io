Posted on 2019-12-16 Edited on 2020-01-08 In Python

可迭代对象 Iterable：

可以用 for 循环遍历的对象，包括：

  • 集合数据类型，如 list、tuple、dict、set、str等；
  • 生成器 generator，包括“( i for i in range(10) )”生成器和带yield的generator function。
