LeetCode-19-删除链表的倒数第n个节点

发表于 2020-01-04 更新于 2020-01-08 分类于 LeetCode

给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

示例：

给定一个链表: 1->2->3->4->5, 和 n = 2.

当删除了倒数第二个节点后，链表变为 1->2->3->5.

说明：

给定的 n 保证是有效的。

进阶：

你能尝试使用一趟扫描实现吗？

思路:
将所有结点放进一个数组中,直接将数组中倒数第n+1个结点的next指针指向倒数第n-1个结点.
